# MailVoyage Code of Conduct

## Our Commitment

MailVoyage is an open-source project for everyone. We want this community to be respectful, practical, and welcoming to contributors of all backgrounds and experience levels.

We expect all community spaces (issues, PRs, discussions, commits, reviews, and chats about the project) to follow this code.

## Expected Behavior

1. Be respectful and constructive.
2. Assume good intent, ask questions before judging.
3. Give actionable feedback, especially in reviews.
4. Focus on the code and behavior, not the person.
5. Be patient with new contributors.

## Unacceptable Behavior

1. Harassment, hate speech, discrimination, or personal attacks.
2. Threats, intimidation, or doxxing.
3. Repeatedly derailing technical discussions.
4. Sexualized language or unwelcome advances.
5. Publicly sharing private information without consent.

## Maintainer Responsibilities

Maintainers will:

1. Moderate fairly and consistently.
2. Protect contributors from harassment.
3. Prioritize de-escalation and constructive resolution.
4. Take action when this code is violated.

Possible actions include warning, comment removal, temporary restrictions, or permanent ban depending on severity and repeated behavior.

## Reporting

If you experience or witness behavior that violates this code:

1. Use GitHub's built-in report option on the offending content (comment, issue, PR, or discussion).
2. If additional project context is needed, open a Discussion in this repository with a neutral summary and no personal details.

When reporting, include:

1. What happened.
2. Where it happened (link/screenshot if possible).
3. When it happened.
4. Any context that helps with review.

Reports will be handled as confidentially as possible.

## Scope

This Code of Conduct applies to:

1. This repository and wiki.
2. Official project discussions and PR reviews.
3. Any space where someone is representing MailVoyage.

## Final Note

Strong technical disagreement is normal. Disrespect is not.

We can be direct and still be kind.
